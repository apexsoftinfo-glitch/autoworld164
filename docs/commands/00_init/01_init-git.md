# Intro

Twoim zadaniem jako agenta AI jest zrealizowanie poniższych poleceń.

CEL: Utworzenie nowego git repository

# Task

1. Upewnij się, że jesteś w folderze z projektem Flutter.
2. Wykonaj `git init`
3. `git add .`
4. `git commit -m "Initial commit"`

Gdy skończysz, poinformuj użytkownika o tym co zrobiłeś.

Jeżeli nie masz uprawnień, poproś użytkownika, aby to sam zrobił dając mu dokładne instrukcje co ma wpisać i gdzie. Zakładaj, że użytkownik jest osobą nietechniczną.

Gdy odpowie "next", przejdź do wykonania polecenia zawartego w `docs/commands/00_init/02_init-env-file.md`.
