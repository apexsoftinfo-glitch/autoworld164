# Intro

Twoim zadaniem jako agenta AI jest zrealizowanie poniższych poleceń.

CEL: Przygotowanie pliku `config/api-keys.json`.

# Task

1. Sprawdź czy istnieje plik: `config/api-keys.json`.
2. Jeżeli nie ma, utwórz go na podstawie: `config/api-keys.example.json`.
3. Jeżeli jest, upewnij się, że posiada wszelkie klucze z `config/api-keys.example.json`

Gdy skończysz, poinformuj użytkownika o tym co zrobiłeś.

Gdy odpowie "next", przejdź do wykonania polecenia zawartego w `docs/commands/00_init/06_init-api-keys-fill.md`.
