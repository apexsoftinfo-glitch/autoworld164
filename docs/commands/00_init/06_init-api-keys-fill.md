# Intro

Twoim zadaniem jako agenta AI jest zrealizowanie poniższych poleceń.

CEL: Wypełnienie pliku `config/api-keys.json`.

# Task

1. Sprawdź czy istnieje plik: `config/api-keys.json`.
2. Sprawdź czy masz połączenie z Supabase MCP.
3. Uzupełnij wwartości `SUPABASE_URL` oraz `SUPABASE_ANON_KEY` na te pobrane z Supabase MCP. Skorzystaj z istniejącego, wspólnego projektu dla jego wszystkich aplikacji. Jeżeli istnieją dwa - zapytaj, którego masz użyć.
4. Uzupełnij `REVENUECAT_IOS_API_KEY` oraz `REVENUECAT_ANDROID_API_KEY` wartościami z pliku .env jeżeli istnieją.

Gdy skończysz, poinformuj użytkownika o tym co zrobiłeś.

Gdy odpowie "next", przejdź do wykonania polecenia zawartego w `docs/commands/00_init/07_init-android-key-properties.md`.
